Remarks
1. This site contains the programs and program excerpts of Kernighan-Ritchie's famous textbook and is intended to help students in their first C experiments.

2. File naming convention: kr#ext , where kr obviously stands for Kernighan-Ritchie, # indicates the page number in Kernighan-Ritchie's textbook, and ext is an extension, i.e. .c for the C source code, _t.c for a test program, etc. Example: kr020.c, kr039_t.c

3. A makefile is provided for each chapter.

4. Our test programs have been designed with two focuses in mind: simplicity (i.e. just what is needed to test the program excerpts) and flexibility (complete enough to test the program excerpts as exhaustively as possible). So the dialog generated by our test programs will be as minimal as possible and assumes that the user has the source program in mind. Note also that in chapter 2 and following we will often use the function scanf for the input data (a description of this function is given in Section 7.4).
