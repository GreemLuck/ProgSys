Remarks
1. Be aware that the proposed solutions have not been validated (this is in particular true for the most complicated ones). But a lot of them contain very nice C code (i.e. 'krx402.c' contains a very smart proposition for adjoining a conditional test program.)

2. All C codes have been compiled by "gcc" or "gcc -c". In order to succeed, some original C codes have been debugged by adding some include files (like stdlib.h or stdio.h), changing some prototype names (like 'strcat' to 'my_strcat' in solution 5-3), and correcting other miscellaneous syntax errors (like '\s' changed to '\n' in solution 5-13). Finally a Makefile has been added for each chapter.

3. Yet unsolved exercises: 2-9, 4-9, 4-10, 5-12, 5-15 to 5-20, 6-2, 6-6, 7-4, 7-5, 7-7, 8-2, 8-5, 8-7
