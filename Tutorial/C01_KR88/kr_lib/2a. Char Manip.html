2a.   Character Manipulations

Character Count 1.  Count characters in input, using a while loop. The number of characters is printed to the output.  kr018_1.c

Character Count 2.  Same as Character Count 1, but using a for loop.  kr018_2.c

Line Count.  Count lines in input. The number of lines is printed to the output.  kr019.c

Word Count 1.  Count lines, words, and characters in input. The number of lines, words and characters are printed to the output.  kr020.c

Word Count 2.  A variant of kr020.c, with a subtle error.  kr020_bh1.c

Word Count 3.  A variant of kr020.c, with a tricky error.  kr020_bh2.c

Count Digits, White Space and Others.  The output will be something like digit = 0 3 0 0 2 1 0 1 0 0, white space = 11, other = 49 where three digits "1", two digits "4", one digit "5", one digit "7", 11 white spaces and 49 other characters have been entered.  kr022.c

Count Digits, White Spaces and Others (version 2).   The output will be something like digit = 0 3 0 0 2 1 0 1 0 0, white space = 11, other = 49 where three digits "1", two digits "4", one digit "5", one digit "7", 11 white spaces and 49 other characters have been entered. This program is identical to kr022.c (see Chap. 1, p. 22), but is implemented with a switch statement.   kr059.c
