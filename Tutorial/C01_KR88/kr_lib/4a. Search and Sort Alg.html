int binsearch(x, v, n).   Search x in a sorted array v of integers, v[0] <= v[1] <= ../kr. v[n-1], and return the position if x occurs in v, and -1 if not.
- The first test program kr058_t reads the value x and a list of sorted integers, and prints the value of binsearch.
- The second test program kr058_t2 is a variant, where the value x is given as argument in the command line (usage: ./kr058_t2  value_x).
- For fun and in order to test binsearch with a big list, kr058_gen.c provides a simple generator of an ordered list of integers.
