Chapter 6 - Structures

Point and Rectangle Structures.  
The test program kr128-129_t.c reads two points defining a rectangle and prints the two points, their distance and the surface of the rectangle.

makepoint(x,y), addpoint(p1,p2), ptinrect(p, rect).  
The test program kr130_t.c reads two points defining a rectangle and prints their sum, then reads two points and tests if they are inside the rectangle or not.

canonrect(rect).  
The test program kr131_t.c reads two points defining a rectangle and prints the coordinates in canonical form (lower-left, upper-right).

Pointer to Structure 1:   How to access structure variables via pointers.
The program kr131_2.c reads two integers defining a point and prints them.

Pointer to Structure 2:   An alternative notation with the structure operator ->
The program kr131_3.c reads two integers defining a corner of a rectangle and prints them using four different notations (access to structure variables either by name or pointer).

Pointer to Structure 3:   About the high order binding of the structure operators . and ->
The program kr132.c reads an integer and prints it incremented by one.

Keyword Counting 1.

Keyword Counting 2.  
The program kr137.c reads a file and prints the occurrences of C keywords, using pointers instead of array indices.

Size of a structure and alignement problem.  
The program kr138.c illustrates the alignement problem within structures showing that a structure containing a char (1 byte) and an integer (typically 4 bytes on a 32 bits machine) requires 8 bytes.

Word Frequency Count 1.  
The program kr139-143.c reads a text file and prints the occurrences of all words, using a tree data structure.

Table Lookup:   Two classical primitives are provided:
  - install(n,t) records the strings n (name) and t (replacement text) in a table, and
  - lookup(n) searches for n in the table.
The test program kr144-145_t.c allows interactively to install n and t (both restricted to words) and lookup n.

Word Frequency Count 2.  
The program kr146.c reads a text file and prints the occurrences of all words, this time using typedef for the tree nodes.

The test program kr148_t.c reads a character, integer, float and word and prints their address, size and value.
