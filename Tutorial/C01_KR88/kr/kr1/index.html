Chapter 1 - A Tutorial Introduction

Hello World.  Display "hello world".  kr006.c

Fahrenheit 1.  Print Fahrenheit-Celsius table for fahr = 0, 20, ..., 300.  kr009.c

Fahrenheit 2.  Same as Fahrenheit 1, but in floating point.  kr012.c

Fahrenheit 3.  Same as Fahrenheit 2, but using a for loop.  kr013.c

Fahrenheit 4.  Same as Fahrenheit 3, but using symbolic constants.  kr015.c

File Copy 1.  Copy input to output.  kr016.c

File Copy 2.  Same as File Copy 1, using a variant of the while loop.  kr017.c

Character Count 1.  Count characters in input, using a while loop. The number of characters is printed to the output.  kr018_1.c

Character Count 2.  Same as Character Count 1, but using a for loop.  kr018_2.c

Line Count.  Count lines in input. The number of lines is printed to the output.  kr019.c

Word Count 1.  Count lines, words, and characters in input. The number of lines, words and characters are printed to the output.  kr020.c

Word Count 2.  A variant of kr020.c, with a subtle error.  kr020_bh1.c

Word Count 3.  A variant of kr020.c, with a tricky error.  kr020_bh2.c

Count Digits, White Space and Others.  The output will be something like digit = 0 3 0 0 2 1 0 1 0 0, white space = 11, other = 49 where three digits "1", two digits "4", one digit "5", one digit "7", 11 white spaces and 49 other characters have been entered.  kr022.c

int power(int b, int n)  (version 1).  Raise base b to power n. The program kr024 displays a table of n, 2**n and -3**n, for n = 0, 1, ...,9.  kr024.c

power(b, n)  (old-style version 1).  Same as version1, but use an earlier version of how functions are declared and defined.  kr026.c

int power(int b, int n)  (version 2).  Same as version1, but slightly optimized (using the fact that in C all function arguments are passed "by value").  kr027.c

Longest Line 1.  Print longest input line.  kr029.c

int getline(s[], n)  (version 1):   getline reads the next n-1 characters into the string s, stopping if a newline is encountered. Note that getline returns the length of the readed string s, or zero on end of file.
The test program kr029_2_t.c reads a file from the standard input, chunk by chunk, and prints them to the standard output with their length.   kr029_2_t.c

Longest Line 2.  Same as 'Longest Line 1'; version with external variables.  kr032.c

Reference: Brian W. Kernighan, Dennis M. Ritchie: The C Programming Language, 2nd Edition, Prentice Hall 1988
