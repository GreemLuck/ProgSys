File Access:  Concatenation of n files.
The program kr162.c reads n files in input given as arguments in the command line, concatenates them and prints the result to the standard output. If no arguments are given in the command line, the standard input is readed and copied line per line to the standard output.
Synopsis:  kr162   [ FILE ]

Error Handling - sdterr and exit.  
The program kr163.c is a revised version of kr162.c, using stderr for error treatment.

char *fgets(*s, n, *f),   int fputs(*s, *f):   fgets reads the next n-1 characters into the string s, stopping if a newline is encountered and fputs writes the string s on stream f. Note that fgets returns NULL on end of file or error.
The program kr165.c copies an input file to an output file.
