4b.   Mathematical Functions

int power(int b, int n)  (version 1).  Raise base b to power n. The program kr024 displays a table of n, 2**n and -3**n, for n = 0, 1, ../kr.,9.  kr024.c

power(b, n)  (old-style version 1).  Same as version1, but use an earlier version of how functions are declared and defined.  kr026.c

int power(int b, int n)  (version 2).  Same as version1, but slightly optimized (using the fact that in C all function arguments are passed "by value").  kr027.c

int rand(),  void srand(seed).   Pseudo-random number generator.
The test program kr046_t.c reads two integers s and n, and generates and prints a series of n pseudo-random numbers with seed s.   kr046.c   kr046_t.c

int rand(),  void srand(seed)  (version 2):   Random Number Generator.
The test program kr168_t.c reads two integers s and n, and generates and prints a series of n pseudo-random numbers between 0 and 1 with seed s.
