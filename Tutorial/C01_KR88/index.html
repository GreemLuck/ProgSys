Labs: Mini Test Environment (MTE)

MTE allows students to experiment miscellaneous C codes:

  • kr contains all programs and program excerpts discussed in [KR 88]
  • krx contains answers to most [KR 88]'s exercises
  • kr_lib is a thematic recompilation of [KR 88] programs and program excerpts
  • adt illustrates how to write Abstract Data Types in C
  • extras contains miscellaneous C Codes and Unix scripts

Reference:
[KR 88] Brian W. Kernighan, Dennis M. Ritchie: The C Programming Language, 2nd Edition, Prentice Hall 1988
