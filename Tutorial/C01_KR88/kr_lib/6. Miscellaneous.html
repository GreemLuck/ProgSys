6.   Miscellaneous

unsigned getbits(unsigned x, int p, int n).   Return the (right adjusted) n-bit field of x that begins at position p. For example, getbits(x,4,3) returns the three bits in bit positions 4, 3, 2, right adjusted. It is assumed that bit position 0 is at the right end and that n and p are sensible positive values.
The test program kr049_t.c reads the three integers x, p, and n and prints the value returned by getbits(x, p, n).   kr049.c   kr049_t.c

int bitcount(unsigned x).   Return the number of 1 bits in its integer argument x. For example, bitcount(7) returns 3.
The test program kr050_t.c reads the unsigned x and prints the value returned by bitcount(x).   kr050.c   kr050_t.c
